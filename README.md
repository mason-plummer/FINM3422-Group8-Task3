# FINM3422-Group8-Task3
This repository is Group 8's official repository in the completion of assessment Task 3 for the course code FINM3422. The purpose of the assignment is to build a prototype risk and derivatives modelling platform. In this repository there are three scripts, each a module responsible for the construction of python code to complete the following tasks: 

1. Building a yield curve
2. Price derivatives
3. Compute portfolio risk metrics

In which, the results from these modules will be transferred to the notebook, with the purpose of explaining the context behind the module codes and the explanation behind the results.
